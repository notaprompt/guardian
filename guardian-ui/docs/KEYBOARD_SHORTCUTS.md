# Guardian Keyboard Shortcuts

Quick reference for all keyboard shortcuts in Guardian.

---

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+1` | Focus Terminal panel |
| `Ctrl+2` | Focus Chat panel |
| `Ctrl+3` | Focus Notes panel |
| `Ctrl+4` | Focus Artifacts/Search panel |
| `Ctrl+Shift+P` | Open Command Palette |
| `Ctrl+Shift+M` | Maximize/restore focused panel |
| `Ctrl+Shift+B` | Create backup |
| `Ctrl+,` | Open Settings |
| `Escape` | Blur active element / close overlays |

---

## Chat

| Shortcut | Action |
|----------|--------|
| `Enter` | Send message |
| `Shift+Enter` | Insert newline in message |

---

## Terminal

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+D` | Split — add a new terminal pane (max 4) |
| `Ctrl+Shift+W` | Close the active terminal pane |
| `Ctrl+Shift+S` | Snapshot active terminal viewport |
| `Ctrl+Tab` | Cycle between terminal panes |
| `Ctrl+V` | Paste from clipboard into terminal |

---

## Command Palette

Open with `Ctrl+Shift+P`, then:

| Shortcut | Action |
|----------|--------|
| `Up / Down` | Navigate results |
| `Enter` | Execute selected command |
| `Escape` | Close palette |

---

## Settings Panel

| Shortcut | Action |
|----------|--------|
| `Ctrl+,` | Toggle settings open/closed |
| `Escape` | Close settings |

---

## Command Palette Commands

All commands available via `Ctrl+Shift+P`:

### Navigation

| Command | Shortcut | Description |
|---------|----------|-------------|
| Focus Terminal | `Ctrl+1` | Switch focus to the terminal panel |
| Focus Chat | `Ctrl+2` | Switch focus to the chat panel |
| Focus Notes | `Ctrl+3` | Switch focus to the notes panel |
| Focus Search | `Ctrl+4` | Switch focus to the artifacts/search panel |

### Session

| Command | Description |
|---------|-------------|
| New Chat Session | Start a fresh conversation |

### Notes

| Command | Shortcut | Description |
|---------|----------|-------------|
| New Scratch Note | `Ctrl+N` | Quick unstructured capture |
| New Structured Note | `Ctrl+Shift+N` | Titled, organized note |
| New Journal Entry | | Date-stamped reflection |

### Memory

| Command | Shortcut | Description |
|---------|----------|-------------|
| Search Memory | `Ctrl+K` | Search conversations, notes, artifacts |

### Layout

| Command | Shortcut | Description |
|---------|----------|-------------|
| Maximize Focused Panel | `Ctrl+Shift+M` | Expand current panel, collapse others |

### Settings

| Command | Shortcut | Description |
|---------|----------|-------------|
| Open Settings | `Ctrl+,` | Configure Guardian preferences |

### Backup & Data

| Command | Shortcut | Description |
|---------|----------|-------------|
| Create Backup | `Ctrl+Shift+B` | Back up all Guardian data |
| Restore from Backup | | Restore from a .zip or .tar.gz backup file |
| Export Current Session | | Export active conversation as Markdown |
| Export Session as JSON | | Export active conversation with full data fidelity |
| Export All Notes | | Export all notes as Markdown files to a directory |
| Export Full Data (JSON) | | Export all sessions, notes, and usage data |
| Import Notes | | Import Markdown files as notes |
| Import Obsidian Vault | | Import notes from an Obsidian vault folder |

---

## Notes

- On macOS, substitute `Cmd` for `Ctrl` in most shortcuts.
- The command palette uses fuzzy matching — you do not need to type the full command name.
- Most recently used commands appear first when the palette opens with no query.
